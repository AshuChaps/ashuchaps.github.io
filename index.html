Apple Music Data Analysis

In this project, I conducted an in-depth data analysis using a 2020 Apple Music Play activity CSV dataset sourced directly from an Apple ID user account. Through refining the dataset for optimal relevance, I ensured the project's message was conveyed effectively. By harnessing the power of PySpark, I seamlessly merged SQL functions with Python code, leveraging the structured data format to enable thorough analysis. Demonstrating proficiency in data manipulation and analysis, I used PySpark's SQL capability to pinpoint the most played artist of 2020. To visually communicate significant insights, I employed Matplotlib for data visualization and integrated Pandas for data management, further amplifying the project's impact.
